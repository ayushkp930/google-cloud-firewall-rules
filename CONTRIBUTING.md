# Contributing Guidelines

Thank you for your interest in this Google Cloud firewall lab project!

## Project Goal

This repository shows how to:

- View existing VPC firewall rules
- Create allow and deny rules for HTTP/SSH
- Test VM access from the internet
- Analyze allowed and denied traffic using Logs Explorer

Contributions are welcome if they:

- Improve explanations or diagrams
- Add safer or clearer firewall rule examples
- Fix typos, formatting, or structure
- Suggest better security best practices

## What Not to Contribute

Please do **NOT** submit:

- Real project IDs, IP addresses, or domain names
- Production firewall configurations
- Any sensitive company or customer information

Such content will be removed.

## How to Contribute

1. Fork this repository.
2. Create a new branch for your changes.
3. Make your edits or additions.
4. Open a Pull Request with:
   - A clear title
   - A short explanation of what you changed and why
